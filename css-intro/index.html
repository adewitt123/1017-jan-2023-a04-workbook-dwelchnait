CSS Coding styles

Browser Defaults you have not supplied any css code
Inline This is for a single specific element (tag) instance. The element contains a parameter called sytle="...." style="rule; [rule;....]" style="color:red; font-weight:bold;"
Embedded Coded in your head area of the web page. Is started with the style element and contains the selectors and the rules for the desired CSS. Applies ONLY to the phyiscal page where the code is placed. Affects ALL selector instances of that type on the page. <style>
selector {rule; [rule;...]}
selector {rule; [rule;...]}
</style>		 <style>
p {font-size:larger; font-weight:bold;}
blockquote {background-color:lightblue;}
</style>
StyleSheet Similar to embedded however, the selectors and rules are code in a separate phyiscal file. This file can be shared throughout the entire web site. This file CAN be SAHARED across the web. Thus, all css is in one common place. This is the preferred technique to use. To use the style sheet on a particular web page, you must reference the stylesheet using a &<link> tag. This link tag needs to be in the head area. selector {rule; [rule;...]}
